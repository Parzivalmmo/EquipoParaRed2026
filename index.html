Red Sucks Guide [cite: 1, 4, 5]

Estrategias y rotaciones optimizadas por regiones para enfrentar a Red y Líderes[cite: 7, 30, 51, 64, 81].

Rival: KANTO [cite: 7]

CHARIZARD: Lanzallamas, Tajo Aéreo, Día Soleado [cite: 8]
ESPEON: Bola Sombra, Psíquico, Deseo [cite: 8]
VENUSAUR: Meteorobola, Bomba Lodo, Pantalla Luz [cite: 9]
BLASTOISE: Salpicar, Llovizna [cite: 9]
SNORLAX: Golpe Cuerpo, Terremoto, Descanso, Reflejo [cite: 10]
PIKACHU: Rayo, Maquinación [cite: 10]
Caso 1 [cite: 16]

Si sale CHARIZARD pegarle y sacrificar Espeon (Pantalla Luz). Si cambia a Espeon tirar Psíquico. Entrar con Honchkrow GOLPE BAJO. Frente a Blastoise dos GOLPE BAJO. GOLPE BAJO a Pikachu / Charizard. FUERZA BRUTA a Snorlax. GOLPE BAJO a Venusaur y luego Salamence VUELO[cite: 17].

Caso 2 [cite: 18]

Si sale SNORLAX / Reflejo / Psíquico (x2). Entrar con Salamence / Danza Dragón / VUELO. Si te saca Blastoise: Danza Dragón / Garra Dragón (x2) / Terremoto a Pikachu / Garra Dragón a Chari, Espeon y Snorlax (si tiene más de la mitad de vida tirar Vuelo) / Vuelo a Venusaur[cite: 19].

Caso 3 [cite: 22]

LUEGO DE BOOSTEAR CON HONCH[cite: 20]:

  • Vuelo si no tiene sol[cite: 21].
  • Si tiene sol y estás a +1: GOLPE BAJO[cite: 21].
  • Si estás a +2: FUERZA BRUTA / GOLPE BAJO[cite: 23, 24].
Caso 4 [cite: 26]

Si sale PIKACHU cambiar a Garchomp / Terremoto. Si te saca Charizard cambiar y sacrificar a Accelgor / Poder Oculto Hielo. Entra con Honch / GOLPE BAJO. Frente a Blastoise GOLPE BAJO (x2). Luego ver los movimientos al boostear Honch[cite: 25].

Caso 5 [cite: 28]

Frente a BLASTOISE tirar Psíquico y sacrificar para entrar con Honch / GOLPE BAJO dos veces. Frente a Charizard GOLPE BAJO. Salamence Danza Dragón / Garra Dragón / VUELO a Venusaur. Frente a Snorlax GARRA DRAGÓN / Accelgor con sacrificio / Garchomp (Terremoto a Espeon y Pikachu)[cite: 27].

Caso Extra: Frente a Espeon

Frente a ESPEON / Pantalla Luz / Reflejo / Psíquico. Entrar con Salamence / Danza Dragón / Garra Dragón. Si te saca Snorlax: tirar Vuelo / si usa Descanso, Vuelo hasta derrotarlo / si te mata, entrar con Honchkrow para rematar con Tajo Umbrío (revisar % de vida de Snorlax antes de pegarle). Luego frente a Blastoise GOLPE BAJO (x2). Frente a Pikachu GOLPE BAJO y a Chari. GOLPE BAJO a Venusaur (si resiste, rematar con movimiento eficaz)[cite: 29].

Rival: TESELIA [cite: 30]

SAMUROTT: Surf, Rayo Hielo, Bloqueo [cite: 31]
EXCADRILL: Tijera X, Terremoto, Cabeza de Hierro, Tormenta Arena [cite: 32]
EMBOAR: Envite Ígneo, Rugido [cite: 32]
HYDREIGON: Cometa Draco, Ida y Vuelta, Llamarada, Despejar [cite: 33]
SERPERIOR: Lluevehojas, Drenadoras, Pulso Dragón, Sustituto [cite: 33]
PIKACHU: Rayo, Poder Oculto, Maquinación [cite: 33]
Caso 1 [cite: 34]

Si sale SERPERIOR meter Pantalla Luz y Reflejo, luego en turno limpio con Accelgor tirar ZUMBIDO. Cambiará a Emboar u otro. Si no es Salamence, cambiar a Salamence / tirar dos Danza Dragón / Terremoto / Garra Dragón a Samurott e Hydreigon / Terremoto a Pikachu y Excadrill[cite: 35].

Caso 2 [cite: 36]

Si sale HYDREIGON / Pantalla Luz / Reflejo / Sacrificar frente a Pikachu. Accelgor Zumbido. Frente a Emboar, si saca a otro que no sea Salamence, cambiar a Salamence[cite: 37]. Si saca a Salamence tirar Danza Dragón (x2) y Garra Dragón frente a Hydreigon / Garra Dragón a Samurott[cite: 38].

Caso 3 [cite: 39]

Si sale EXCADRILL / Cambiar consecutivamente a Ferro / Honch / Ferro / Honch / Ferro (se mata solo con Casco Dentado). Frente a Emboar u otro, sacrificar a Ferro. Tirar Espeon con Reflejo / Bostezo / Entrar con Salamence / dos Danza Dragón / Terremoto. Contra Samurott e Hydreigon usar Garra Dragón / Pikachu Terremoto / Serperior VUELO y Garra[cite: 40].

Caso 4 [cite: 41]

Si sale SAMUROTT con Espeon / Pantalla Luz y Reflejo / luego Psíquico. Si sale Hydreigon: Sacrificar / Bostezo. Si saca Serperior: Zumbido de Accelgor. Si cambia a Emboar y no está Salamence, sacrificar a Accelgor. Entrar con Salamence / Danza Dragón (x2) / Terremoto a Emboar, Pikachu y Excadrill / Garra Dragón a Hydreigon[cite: 42].

Caso 5 [cite: 43]

Si sale PIKACHU: T1 Cambiar a Garchomp / T2 Terremoto / T3 Cambiar a Espeon (si sobrevive tirar Pantalla Luz y Reflejo). Si cambia a Serperior, sacrificar Espeon / Accelgor Zumbido. Frente a Emboar si saca otro, sacrificar a Accelgor. Entrar con Salamence / Danza Dragón (x2) / Terremoto a Emboar y Excadrill / Garra Dragón a Samurott e Hydreigon[cite: 44].

Notas Importantes para Teselia[cite: 45]:

  • Accelgor solo sirve para eliminar a Serperior o Hydreigon, luego usar de sacrificio limpio para Salamence[cite: 46].
  • Con vida completa: Danza Dragón y Garra Dragón. Si es el último Pokémon, golpear directamente[cite: 47].
  • Con Reflejo activo: Usar Danza Dragón (x2) y Terremoto[cite: 48].
  • Si no te dejan boostear con Salamence, guardar siempre para Excadrill haciendo pivot entre Honchkrow y Ferrothorn[cite: 50].

Caso 1 [cite: 88]

Frente a INFERNAPE: Pantalla Luz (cambia a Empoleon) / Bostezo / Reflejo. Al dormir, cambiar a Salamence[cite: 89]. Si entra Torterra: VUELO[cite: 90, 107, 114]. Si Infernape cambia a Garchomp: Terremoto si pega Puño Trueno. Entrar con Salamence ante Empoleon: Terremoto hasta ganarle. Ante Staraptor: Sacrificar Espeon, meter Ferrothorn con Casco Dentado frente a Treparrocas[cite: 90, 106]. Frente a Garchomp: Honchkrow GOLPE BAJO / Accelgor ZUMBIDO[cite: 90].
Si saca Pikachu: Sacrificar Espeon -> Accelgor ZUMBIDO -> Sacrificar Garchomp -> Honch GOLPE BAJO -> Sacrificar Ferro/Sala -> Rematar con Golpe Bajo a Staraptor[cite: 91].

Caso 2 [cite: 92]

Si inicia con PIKACHU: Cambiar a Garchomp Golpe Bis. Colocar Honchkrow GOLPE BAJO. GOLPE BAJO a Garchomp. Espeon Psíquico. Cambiar a Salamence / Vuelo hasta derrotar. Frente a Garchomp pivotar Espeon -> Ferrothorn -> Espeon Psíquico. Si cambia a Empoleon: Pantalla Luz / Bostezo / Reflejo -> Salamence TERREMOTO. Frente a Garchomp meter Ferro y sacrificar ante Infernape. Entrar Espeon con Psíquico y a Staraptor / Rematar con Accelgor[cite: 93].

Caso 3 [cite: 94]

Si inicia con TORTERRA: Cambiar a Ferrothorn -> Cambiar a Salamence / Vuelo hasta debilitar. Si saca Staraptor, poner siempre en frente a Ferrothorn para dañarlo con el Casco Dentado. Entrar Espeon ante Empoleon: Reflejo / Bostezo / Pantalla Luz -> Salamence Terremoto. Contra Pikachu: Garchomp Golpe Bis -> Honch GOLPE BAJO. Si saca Garchomp: Espeon Psíquico e Infernape -> Finalizar con Accelgor[cite: 95].

Caso 4 [cite: 96]

Si inicia con EMPOLEON: Pantalla Luz / Bostezo / Reflejo[cite: 97, 101, 102]. Cambiar a Salamence / Danza Dragón / Terremoto / Vuelo a Torterra / Garra Dragón frente a Infernape. Si cae Salamence: Honchkrow GOLPE BAJO. Frente a Pikachu: Garchomp Golpe Bis -> Honchkrow GOLPE BAJO. Contra Garchomp: GOLPE BAJO -> Rematar Espeon Psíquico y Staraptor junto al daño de Ferrothorn[cite: 97].

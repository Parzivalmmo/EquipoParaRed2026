Red Sucks Guide

Estrategias y rotaciones optimizadas por regiones para enfrentar a Red y Líderes.

Rival: KANTO

CHARIZARD: Lanzallamas, Tajo Aéreo, Día Soleado
ESPEON: Bola Sombra, Psíquico, Deseo
VENUSAUR: Meteorobola, Bomba Lodo, Pantalla Luz
BLASTOISE: Salpicar, Llovizna
SNORLAX: Golpe Cuerpo, Terremoto, Descanso, Reflejo
PIKACHU: Rayo, Maquinación
Caso 1

Si sale CHARIZARD pegarle y sacrificar Espeon (Pantalla Luz). Si cambia a Espeon tirar Psíquico. Entrar con Honchkrow GOLPE BAJO. Frente a Blastoise dos GOLPE BAJO. GOLPE BAJO a Pikachu / Charizard. FUERZA BRUTA a Snorlax. GOLPE BAJO a Venusaur y luego Salamence VUELO.

Caso 2

Si sale SNORLAX / Reflejo / Psíquico (x2). Entrar con Salamence / Danza Dragón / VUELO. Si te saca Blastoise: Danza Dragón / Garra Dragón (x2) / Terremoto a Pikachu / Garra Dragón a Chari, Espeon y Snorlax (si tiene más de la mitad de vida tirar Vuelo) / Vuelo a Venusaur.

Caso 3

LUEGO DE BOOSTEAR CON HONCH:

  • Vuelo si no tiene sol.
  • Si tiene sol y estás a +1: GOLPE BAJO.
  • Si estás a +2: FUERZA BRUTA / GOLPE BAJO.
Caso 4

Si sale PIKACHU cambiar a Garchomp / Terremoto. Si te saca Charizard cambiar y sacrificar a Accelgor / Poder Oculto Hielo. Entra con Honch / GOLPE BAJO. Frente a Blastoise GOLGE BAJO (x2). Luego ver los movimientos al boostear Honch.

Caso 5

Frente a BLASTOISE tirar Psíquico y sacrificar para entrar con Honch / GOLPE BAJO dos veces. Frente a Charizard GOLPE BAJO. Salamence Danza Dragón / Garra Dragón / VUELO a Venusaur. Frente a Snorlax GARRA DRAGÓN / Accelgor con sacrificio / Garchomp (Terremoto a Espeon y Pikachu).

Caso Extra: Frente a Espeon

Frente a ESPEON / Pantalla Luz / Reflejo / Psíquico. Entrar con Salamence / Danza Dragón / Garra Dragón. Si te saca Snorlax: tirar Vuelo / si usa Descanso, Vuelo hasta derrotarlo / si te mata, entrar con Honchkrow para rematar con Tajo Umbrío (revisar % de vida de Snorlax antes de pegarle). Luego frente a Blastoise GOLPE BAJO (x2). Frente a Pikachu GOLPE BAJO y a Chari. GOLPE BAJO a Venusaur (si resiste, rematar con movimiento eficaz).

Rival: TESELIA

SAMUROTT: Surf, Rayo Hielo, Bloqueo
EXCADRILL: Tijera X, Terremoto, Cabeza de Hierro, Tormenta Arena
EMBOAR: Envite Ígneo, Rugido
HYDREIGON: Cometa Draco, Ida y Vuelta, Llamarada, Despejar
SERPERIOR: Lluevehojas, Drenadoras, Pulso Dragón, Sustituto
PIKACHU: Rayo, Poder Oculto, Maquinación
Caso 1

Si sale SERPERIOR meter Pantalla Luz y Reflejo, luego en turno limpio con Accelgor tirar ZUMBIDO. Cambiará a Emboar u otro. Si no es Salamence, cambiar a Salamence / tirar dos Danza Dragón / Terremoto / Garra Dragón a Samurott e Hydreigon / Terremoto a Pikachu y Excadrill.

Caso 2

Si sale HYDREIGON / Pantalla Luz / Reflejo / Sacrificar frente a Pikachu. Accelgor Zumbido. Frente a Emboar, si saca a otro que no sea Salamence, cambiar a Salamence. Si saca a Salamence tirar Danza Dragón (x2) y Garra Dragón frente a Hydreigon / Garra Dragón a Samurott.

Caso 3

Si sale EXCADRILL / Cambiar consecutivamente a Ferro / Honch / Ferro / Honch / Ferro (se mata solo con Casco Dentado). Frente a Emboar u otro, sacrificar a Ferro. Tirar Espeon con Reflejo / Bostezo / Entrar con Salamence / dos Danza Dragón / Terremoto. Contra Samurott e Hydreigon usar Garra Dragón / Pikachu Terremoto / Serperior VUELO y Garra.

Caso 4

Si sale SAMUROTT con Espeon / Pantalla Luz y Reflejo / luego Psíquico. Si sale Hydreigon: Sacrificar / Bostezo. Si saca Serperior: Zumbido de Accelgor. Si cambia a Emboar y no está Salamence, sacrificar a Accelgor. Entrar con Salamence / Danza Dragón (x2) / Terremoto a Emboar, Pikachu y Excadrill / Garra Dragón a Hydreigon.

Caso 5

Si sale PIKACHU: T1 Cambiar a Garchomp / T2 Terremoto / T3 Cambiar a Espeon (si sobrevive tirar Pantalla Luz y Reflejo). Si cambia a Serperior, sacrificar Espeon / Accelgor Zumbido. Frente a Emboar si saca otro, sacrificar a Accelgor. Entrar con Salamence / Danza Dragón (x2) / Terremoto a Emboar y Excadrill / Garra Dragón a Samurott e Hydreigon.

Notas Importantes para Teselia:

  • Accelgor solo sirve para eliminar a Serperior o Hydreigon, luego usar de sacrificio limpio para Salamence.
  • Con vida completa: Danza Dragón y Garra Dragón. Si es el último Pokémon, golpear directamente.
  • Con Reflejo activo: Usar Danza Dragón (x2) y Terremoto.
  • Si no te dejan boostear con Salamence, guardar siempre para Excadrill haciendo pivot entre Honchkrow y Ferrothorn.

Rival: HOENN

SWAMPERT: Terremoto, Puño Hielo, Escaldar, Trampa Rocas
BLAZIKEN: Envite Ígneo, Agilidad
SALAMENCE: Garra Dragón, Colmillo Ígneo, Danza Dragón
PIKACHU: Rayo, Carga
METAGROSS: Descanso, Puño Hielo, Terremoto, Bloqueo
SCEPTILE: Pulso Dragón, Sustituto
Caso 1

Si sale SWAMPERT: T1 Reflejo / T2 Pantalla Luz / T3 Cambiar a Ferrothorn / T4 Latigazo (si Swampert quema a Ferro, tirar Latigazo otra vez). Debería sacar a Blaziken. Sacrificar a Ferrothorn / Entrar con HONCHKROW / GOLPE BAJO. Frente a Metagross tirar Vuelo dos veces (asegurar los impactos) y Tajo Umbrío hasta derrotarlo.

Rival: JOHTO

GLISCOR: Terremoto, Respiro, Ida y Vuelta, Trampa Rocas
TYRANITAR: Terremoto, Puño Hielo, Triturar, Onda Trueno
TYPHLOSION: Estallido, Avivar
FERALIGATR: Puño Hielo, Danza Dragón
MEGANIUM: Drenadoras, Gigadrenado, Pantalla Luz
PIKACHU: Rayo, Agilidad
Caso 1
  • T1: Reflejo
  • T2: Cambiar a Ferrothorn
  • T3: Imagen
  • T4: Cambiar a Espeon
  • T5: Colocar Pantalla Luz
  • T6: Repetir ciclo hasta morir

Fase de Limpieza:

Meter Salamence / Danza Dragón (x2) (+2 Atk / +2 Speed) y barrer con TERREMOTO o GARRA DRAGÓN.

Rival: SINNOH

STARAPTOR: Pájaro Osado, A Bocajarro, Remolino
EMPOLEON: Rayo Hielo, Escaldar, Despejar
TORTERRA: Hoja Aguda, Roca Afilada, Terremoto, Reflejo
INFERNAPE: A Bocajarro, Puño Fuego, Danza Espada (Banda Focus)
GARCHOMP: Carga Dragón, Terremoto, Afilagarras
PIKACHU: Rayo, PO Dragón, Sustituto

Nota: Salamence es más lento que Garchomp, Infernape, Staraptor y Pikachu por defecto.

Caso 1

Frente a INFERNAPE: Pantalla Luz (cambia a Empoleon) / Bostezo / Reflejo. Al dormir, cambiar a Salamence. Si entra Torterra: VUELO. Si Infernape cambia a Garchomp: Terremoto si pega Puño Trueno. Entrar con Salamence ante Empoleon: Terremoto hasta ganarle. Ante Staraptor: Sacrificar Espeon, meter Ferrothorn con Casco Dentado frente a Treparrocas. Frente a Garchomp: Honchkrow GOLPE BAJO / Accelgor ZUMBIDO.
Si saca Pikachu: Sacrificar Espeon -> Accelgor ZUMBIDO -> Sacrificar Garchomp -> Honch GOLPE BAJO -> Sacrificar Ferro/Sala -> Rematar con Golpe Bajo a Staraptor.

Caso 2

Si inicia con PIKACHU: Cambiar a Garchomp Golpe Bis. Colocar Honchkrow GOLPE BAJO. GOLPE BAJO a Garchomp. Espeon Psíquico. Cambiar a Salamence / Vuelo hasta derrotar. Frente a Garchomp pivotar Espeon -> Ferrothorn -> Espeon Psíquico. Si cambia a Empoleon: Pantalla Luz / Bostezo / Reflejo -> Salamence TERREMOTO. Frente a Garchomp meter Ferro y sacrificar ante Infernape. Entrar Espeon con Psíquico y a Staraptor / Rematar con Accelgor.

Caso 3

Si inicia con TORTERRA: Cambiar a Ferrothorn -> Cambiar a Salamence / Vuelo hasta debilitar. Si saca Staraptor, poner siempre en frente a Ferrothorn para dañarlo con el Casco Dentado. Entrar Espeon ante Empoleon: Reflejo / Bostezo / Pantalla Luz -> Salamence Terremoto. Contra Pikachu: Garchomp Golpe Bis -> Honch GOLPE BAJO. Si saca Garchomp: Espeon Psíquico e Infernape -> Finalizar con Accelgor.

Caso 4

Si inicia con EMPOLEON: Pantalla Luz / Bostezo / Reflejo. Cambiar a Salamence / Danza Dragón / Terremoto / Vuelo a Torterra / Garra Dragón frente a Infernape. Si cae Salamence: Honchkrow GOLPE BAJO. Frente a Pikachu: Garchomp Golpe Bis -> Honchkrow GOLPE BAJO. Contra Garchomp: GOLPE BAJO -> Rematar Espeon Psíquico y Staraptor junto al daño de Ferrothorn.
